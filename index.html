CPU

A Central Processing Unit is the primary component of a computer. It is responsible for the execution of intructions and performing calculations. It acts like the brain of a computer, processing data and carrying out multiple tasks required by programs. They contain multiple cores, which is what allows them handle those multiple tasks. They also interact with other hardware components.

RAM

Random Access Memory is a type of computer memory that temporarily stores data. Unlike hard drive or SSDs, RAM is volatile, meaning it loses all its data when powered off. RAM enables fast access to data, allowing faster retrieval and manipulation of information by the CPU.

Motherboard

A motherboard is the main circuit board of a computer that houses and connects all the essential componenets. It provides the necessary connections for all the components to communicate and function properly. The motherboard also typically includes ports for connecting external devices.

Cooling Fans

A cooling fan for a PC is a component designed to dissipate heat generated by the CPU, GPU, and other components. They are esstential for maintaing optimal operating temperatures and ensuring the stability and longevity of the computer system.

GPU

A Graphics Processing Unit is a specialized electronic circuit designed to accelerate the creation and rendering of images, videos, and 3D graphics. Unlike the CPU, which is more general purpose, the GPU is optimized for parallel processing tasks related to graphics rendering.

SSDs & Hard Drives

Both SSDs and Hard Drives are used to store data. The main difference is that SSDs do not have magnetic disks, which results in faster data access and lower latency. Hard Drives use the magnetic disks to store large amounts of data, and typically come with larger sizes, and cheaper prices. However, hard drives tend to be slower than SSDs.

Power Supply Unit

A PSU is a critical component of a computer that converts electrical power from an external source into the appropriate voltage and current needed to power the computer. It contains various connectors and cables to distribute power to different components within the PC.

Cases

PC Cases provide the necessary space for all the internal components, and wiring to connect all the pieces together. They allow for better airflow between the components, and also provide the fragile pieces protection.
