CPUs

A CPU is made up of 5 main parts. The ALU, the CU, registers, cache memory, and the clock. The Arithmetic Logic Unit is responsible for executing instructions and performing calculations that are required by software programs. The Control Unit directs the flow of data instructions within the CPU and other hardware components. Registers are high speed memory units that store temporary data, instructions, and memory address used by the ALU and CU during execution. Cache memory stores frequently used data and instructions and reduces time it takes for the CPU to access them. The clock is a timing device that synchronizes the operations of the CPU and other hardware components.

Intel puts more emphasis on higher single-threaded performance and clock speed, while AMD Ryzen tend to offer more cores at competitve prices. The main difference is that most Intel CPUs come with integrated graphics, whereas AMD Ryzen requires you to also own a GPU.
